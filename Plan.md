# AlgoMotion — Week 1 

- Build the absolute foundation of the rendering system and understand how animation engines work internally.

By the end of Week 1, you should be able to:

Open a rendering window
Draw squares
Draw text
Create multiple array cells
Render values inside cells
Understand game/render loops
Understand coordinate systems
Build your first “visual array”
